Parse CompaniesHouse API data and generate dataset.




The purpose of this project is to identify different types of companies all over the UK and generate a dataset to analyse their financial performance over the current and previous year. The final generated dataset can be used to cluster companies based on revenue/performance, predict a company's future performance and so on. This project covers the entire parsing pipeline and converts raw pdf information into clean and usable data.




The application connects to the CompaniesHouse Platform (Gov UK) and fetches the filings documents (PDF files). The files are then parsed for the required information. The time taken depends on the number of pages.

It takes approximately 30-40 seconds to process and extract the data, so please be patient :)



Tech Stack:
Python - Backend processing
Tesseract - Optical Character Recognition

API calls:
REST API calls to CompaniesHouse
REST API calls to Python Backend from Flask
